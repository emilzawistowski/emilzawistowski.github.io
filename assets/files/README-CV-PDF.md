# CV PDF Files

Place your CV PDF files in `/assets/files/`:

```
/assets/files/
├── cv-research.pdf          # Academic CV for research section
├── cv-creative.pdf          # Artistic CV for creative section
└── cv-full.pdf              # Complete CV (optional)
```

## Creating Your CV PDF

You can create these PDFs:
1. From the web page itself (browser print-to-PDF)
2. From a LaTeX CV template
3. From a Word/Pages document
4. From a dedicated CV builder

## Updating CV

When you update your CV:
1. Update the content in `_pages/research/about.md` or `_pages/creative/about.md`
2. Replace the PDF file with the new version
3. Keep the same filename for consistent links

## Alternative: No PDF Download

If you prefer not to offer PDF downloads, simply remove the "Download CV" section from the about pages.
